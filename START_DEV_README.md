# Скрипты запуска для разработки

Скрипты для автоматического запуска backend сервера, frontend и SSH туннеля.

## Использование

### Bash (Git Bash / Linux / macOS)

```bash
./start_dev.sh
```

### PowerShell (Windows)

```powershell
.\start_dev.ps1
```

## Что запускается

1. **Backend сервер** (FastAPI)
   - Адрес: `http://127.0.0.1:5565`
   - Документация API: `http://127.0.0.1:5565/docs`

2. **Frontend** (Vite + React)
   - Адрес: `http://localhost:5174`

3. **SSH туннель**
   - Пробрасывает порт 5565 с удаленного сервера `46.101.158.230` на локальный порт 5565
   - Для использования с другим пользователем SSH, установите переменную окружения:
     ```bash
     export SSH_USER=your_username
     ./start_dev.sh
     ```

## Остановка

Нажмите `Ctrl+C` в терминале, где запущен скрипт. Все сервисы будут автоматически остановлены.

## Логи

Логи сохраняются в файлы:
- `backend.log` - логи backend сервера
- `frontend.log` - логи frontend
- `ssh_tunnel.log` - логи SSH туннеля

## Примечания

- Убедитесь, что порты 5565 и 5174 свободны перед запуском
- Для SSH туннеля требуется настроенный SSH доступ к серверу `46.101.158.230`
- Если SSH туннель не нужен, вы можете закомментировать соответствующий раздел в скрипте
